Custom Background Sync and Push Events

  1. Go to chrome://version and make sure you're running Chrome 64 or higher.
  2. Open DevTools.
  3. Click the Application tab.
  4. Click the Service Workers tab.
  5. Enter text in the Push text box then click Push to send push notifications with custom content.
  6. Enter text in the Sync text box then click Sync to send background sync events with custom tags.
